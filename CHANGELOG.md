# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-19

### New Features
- **Team/Player Color Indicator** - Visual colors to distinguish teams
- **Rally or Sideout Scoring** - Choose between rally scoring (any point) or sideout (only serving team scores)

### Improvements
- Dark mode enhancements and bug fixes
- Theme system refactored for better performance
- Updated scoring logic with improved side-switching
- Color palette improvements

### Bug Fixes
- Fixed undo functionality
- Fixed theme provider errors

### Other
- Privacy policy updated
